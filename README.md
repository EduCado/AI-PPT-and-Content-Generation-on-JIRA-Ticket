Trigger

A new issue is created in Jira for a specific team.
The issue contains predefined fields and content (e.g., campaign type, target audience, product information, key messaging).

Workflow Execution

The workflow monitors Jira for new issues that match the specified team and criteria.
When a qualifying issue is created, the workflow automatically extracts the relevant information.
An AI content-generation service creates marketing materials based on:
Company-approved templates
Brand guidelines
Marketing instructions
Tone and style standards

Content Generation

Marketing copy is generated (email content, campaign summary, social posts, etc.).
A presentation deck (PPT) is automatically created using corporate branding and approved layouts.

Approval Process

The generated PPT and marketing content are routed to designated reviewers.
Approvers receive notifications via email or collaboration tools.
